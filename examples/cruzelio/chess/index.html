This is a collection of chess games designed by LLMs to test the capabilities of different AI models. Welcome to the LLM Arena. Each LLM was given only a single prompt to design a chess game. the prompt was: "I need a single file html application that simulates a chess game in theme of cyberpunk spaghetti western. It should be responsive for a mobile device. Every time a piece is moved, an animation event should queue up. The game should be human vs computer or local 2 player mode where one player moves, the board flips, and another person picks up the same device and makes a move. It should preserve all the rules of traditional chess."
